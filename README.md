# USBCamera
USBカメラを表示するPythonアプリケーション

# 推奨環境
  Windows11

# 必要パッケージ
  |  パッケージ名  |  バージョン  |
  |  ---  |  ---  |
  |  Python  |  3.8以上  |
  |  pip  |  できれば最新  |
  |  OpenCV  |  できれば最新  |
  |  PySide6  |  できれば最新  |

## インストールコマンド
  requirements.txtを使う場合
  ```
  pip install -r requirements.txt
  ```

  直接入れる場合  
  ```
  pip install opencv-python PySide6
  ```
